## Estructuras Condicionales
### Sintaxis de if
```
if(<condición>){
    <sentencias>
}
```
### Sintaxis de if / else
```
if(<condición>){
    <sentencias A>
} else {
    <sentencias B>
}
```
### Sintaxis de if anidado
```
if (<condicion1>) {
    <sentencias A>
} else if(<condicion2>){
    <sentencias B>
} else {
    <sentencias C>
}
```
### Sintaxis de Switch
```
switch(<variable>) {
  case <valor1>:
      <sentencias1>
  break;
  case <valor2>:
      <sentencias2>
  break;
  default:
      <sentencias3>
}
```
## Estructuras Repetitivas
### Sintaxis de While
```
while (<condición>) {
     <sentencias>
}
```
### Sintaxis de Do / while
```
do {
    <sentencias>
} while (<condición>);
```
### Sintaxis de for
```
for (<inicialización>; <terminación>; <incremento>) {
      <sentencias>
}
```
